# üèè IPL Cricket Analytics

##  Project Overview
This project focuses on analyzing Indian Premier League (IPL) cricket data
to uncover player performance trends, team patterns, and match insights
using statistical and visual analytics.

##  Objectives
- Analyze player and team performance
- Identify top run-scorers and wicket-takers
- Study strike rate and economy rate trends
- Apply regression techniques to cricket data

##  Tools & Technologies
- R (ggplot2, dplyr, corrplot)
- Python (Pandas, Matplotlib, Seaborn)
- Jupyter Notebook
- RStudio

##  Key Analysis
- Top batsmen and bowlers across seasons
- Strike rate vs runs relationship
- Team-wise performance comparison
- Correlation heatmaps for performance metrics
- Linear & multiple regression analysis
